# FE-Day11-Service-CRUD-using-HTTP

## Json server
  ```sh
  npm install -g json-server
  ```
  add package.json in scripts
  ```sh
  "json-run":"json-server --watch db.json"
  ```
  and run
  ```sh
  npm run json-run
  ```
